# 有效的字母异位词

异位词：字母及个数都一样，位置不一样

输入: s = "anagram", t = "nagaram"
输出: true

方法1：暴力解法，字母排序sort，再判断字符串是否相等

['c','b','a'].sort()  =>  ["a", "b", "c"] 可以给字母排序
if(s.split('').sort() == t.split('').sort())

方法2：hash，map ---> 统计每个字符的频次 第一个单词，碰到字母就计数+1，第二个单词碰到相同的字母计数-1，最后判断map是否为空

用一个长度为255的（一个ASCLL码最大值0到255），长度为256的数组来计数，相当于简化了的哈希表，只是哈希函数就是ASCLL码的值

统计字符串出现频次

function isAnagram(s,t){
    if(s.length != t.length) return false
    let counter = []
    for(let i=0;i<s.length;i++){
        counter[s.charAt(i)-'a'] ++;
        counter[t.charAt(i)-'a'] --;
    }
    if(counter !=0 ) return false
    return true
}
