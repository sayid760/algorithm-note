# 合并区间

将一组区间中所有重叠的区间进行合并，例如 [起点，终点] 这种方式

输入：[[1,3],[2,6],[15,18],[8,10],[10,11],[7,8]]
输出：[[1,6],[15,18],[7,11]]

思路：排序再判断有没有重叠

  • 1）先排序
  • 2）定义两个变量存放前一个区间的最小值，最大值
  • 3）遍历循环，
    当前区间的最小值小于或等于前一个区间的最大值时，表示有重叠可以合并； 当前区间的最小值大于前一个区间的最大值时，表示不可以合并，并把之前存储的拿出来合并ans.push([start, end])，并保存当前的最小值，最大值
  • 4）遍历循环之后得到的最小值、最大值表示不能被合并的，需要合成区间，并push到数组里
var arr = [[1,3],[2,6],[15,18],[8,10],[10,11],[7,8]]
const merge = arr => {
    // 排序 [[1,3],[2,6],[7,8],[8,10],[10,11],[15,18]]
    arr.sort(function(a, b) {
        if (a[0] !== b[0]){
            return a[0] - b[0];
        }   
        return a[1] - b[1];
    });
    let len = arr.length,
        ans = [],
        start, end; // 用来保存之前的，跟后面的区间做笔记
    for (let i = 0; i < len; i++) {
        let s = arr[i][0],
        e = arr[i][1];
        if (start === undefined){
            start = s, end = e;
        }else if (s <= end){ // 第二个区间的第一s是否比前一个区间的第二位还小，小就说明可以合并 [1,3][2,6]   2比3小
            // 判断当前区间的第二位和前一个区间的第二位数哪个大，取最大的 [1,3][2,6]  3<6 取最大6
            end = Math.max(e, end); 
        }else { // 第二区间的第一位比前一个区间的第二位还大 [2,6],[7,8]   7>6，说明不合并，到这步就不可以判断是否合并了，就把之前的先拿出来合并  
            let part = [start, end];  
            ans.push(part);
            // 保存的是当前的
            start = s;  
            end = e;
        }
    }
    if (start !== undefined) {
        let part = [start, end];
        ans.push(part);
    }
    return ans;
};
console.log(merge(arr))

# 给定一个区间，问最多保留多少个区间，可以让这些区间直接互相不重叠

思路：找出所有子区间的组合，之后判断它不重叠 O((2^n)*n)

借鉴 最长上升子序列 解决问题，

①将全部进行从小到大排列。然后你会发现这题就是最长上升子序列的一个类型题了。直接运用最长上升子序列去解即可。找出其状态转移方程：memo[i]=max(memo[i],memo[j]+1)。 ②贪心算法求解，认为其结尾越小的放在前面计算后面能放更多。
